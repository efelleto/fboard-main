# fBoard - Crie sua Scoreboard
Plugin de criação de Scoreboards de Minecraft baseado no Spigot 1.8.8

### Setup
A dependência utilizada no projeto foi o [Assemble](https://github.com/ThatKawaiiSam/Assemble)
